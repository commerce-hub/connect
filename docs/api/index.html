API

<commerce-hub-connect>

CommerceHub Connect Web SDK widget shell.

Renders SDK-managed shopper flows for CommerceHub interactions, including OTP, saved payment-source selection, source creation, source verification, Pay by Bank handoff, and required-info collection.

Properties

interactionResponse
Type
InteractionResponse

Required CommerceHub interaction payload that controls the visible widget state.

Assign this as a DOM property before shopper interaction, not as an HTML attribute. SDK-managed API calls use interactionResponse.interactionId as the x-interaction-id header.

designConfig
Type
DesignConfig | undefined

Optional merchant-safe colors, typography, spacing, copy, and Pay by Bank provider settings.

Assign this as a DOM property. Omitted nested values fall back to SDK defaults.

baseURL
Attribute
base-url
Type
CommerceHubBaseURLInput | undefined

CommerceHub API base URL preset or custom origin for SDK-managed requests.

Events

onSelect

Emitted with a PaymentSource detail after a verified payment source is attached or selected.

onAddPaymentMethod

Emitted with a PaymentSource detail after a payment source is successfully created.

onVerifyPaymentMethod

Emitted with a PaymentSource detail after a payment source is successfully verified.

onDeletePaymentMethod

Emitted with the deleted payment source id after a payment source is successfully deleted.

onListPaymentMethods

Emitted with a PaymentSource[] detail after payment sources are successfully refreshed from the interaction.

onTransfer

Emitted with a NativeAccountTransaction detail after a native account transfer is successfully initiated.
