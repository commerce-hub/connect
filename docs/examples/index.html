Example: Basic

  • Render the widget and assign an InteractionResponse
  • Bind SDK properties from a Lit host component
  • Create and mount the widget imperatively
  • Customize merchant-safe colors, spacing, and copy

HTML

<commerce-hub-connect></commerce-hub-connect>

JavaScript


import 'commerce-hub-connect-sdk';

const widget = document.querySelector('commerce-hub-connect');

if (widget) {
  widget.interactionResponse = await fetch('/api/commercehub/interaction').then(
    (response) => response.json()
  );
  widget.baseURL = 'QA';
}
